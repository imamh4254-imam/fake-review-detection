# 📊 Fake Review Detection Dashboard

## 📖 Overview
This project analyzes customer reviews to detect **fake vs genuine reviews** and visualize sentiment using Power BI.

## 📊 Dashboard Features
* Sentiment Analysis (Positive, Neutral, Negative)
* Fake vs Genuine Prediction
* Rating distribution
* Review insights

## 🛠️ Tools Used
* Microsoft Power BI
* Dataset (CSV)

## 📂 Files
* `fake_review_dashboard.pbix` → Power BI dashboard

## ▶️ How to Use
1. Download the `.pbix` file
2. Open in Microsoft Power BI Desktop
3. Explore the dashboard
